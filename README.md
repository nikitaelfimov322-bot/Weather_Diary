# Weather Diary

Автор: Никита

## Описание
Простое GUI-приложение на Python для ведения дневника погоды.

## Возможности
- Добавление записей (дата, температура, описание, осадки)
- Фильтрация по дате и температуре
- Сохранение в JSON
- Загрузка данных

## Запуск

Установите Python 3 и выполните:

```bash
python main.py
